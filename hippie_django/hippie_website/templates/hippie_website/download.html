{# ── Format description (TXT + MITAB tabs) ── #}
File formats

Both files are UTF-8, tab-separated, one binary interaction per line, with a header row. Empty or unavailable fields are written as -.

{# ── TXT pane ── #}

HIPPIE-current.txt — compact HIPPIE tab format keyed on UniProt accession. 8 columns; the trailing info column packs all evidence into a single field.

#ColumnExampleDescription
1 uniprot_accession_A P38398 UniProt accession of protein A
2 uniprot_name_A BRCA1_HUMAN UniProt entry name of protein A
3 entrez_A 672 NCBI Entrez Gene ID of protein A
4 uniprot_accession_B P04637 UniProt accession of protein B
5 uniprot_name_B P53_HUMAN UniProt entry name of protein B
6 entrez_B 7157 NCBI Entrez Gene ID of protein B
7 score 0.85 HIPPIE confidence score (0–1), 2 decimals. ≥{{ current_release.int_median|default:0.00|floatformat:2 }} medium, ≥{{ current_release.int_q3|default:0.0|floatformat:2 }} high confidence
8 info experiments:Two-hybrid;pmids:20818382;sources:BioGRID,IntAct Evidence bundle (see below)

info packs three ;-separated groups, each a comma-separated list: experiments:<…>;pmids:<…>;sources:<…>.

{# ── MITAB pane ── #}

HIPPIE-current.mitab.txt — PSI-MI TAB 2.5: 15 mandatory columns plus 3 HIPPIE extension columns (15–18). Within a column, multiple values are joined with |; controlled-vocabulary terms use psi-mi:"MI:xxxx"(name); parts containing reserved characters (| ( ) : or tab) are double-quoted.

#ColumnExampleDescription
1 ID Interactor A entrez gene:672 Primary ID of protein A (Entrez gene)
2 ID Interactor B entrez gene:7157 Primary ID of protein B (Entrez gene)
3 Alt IDs Interactor A uniprotkb:P38398 Alternative ID of protein A (UniProt accession)
4 Alt IDs Interactor B uniprotkb:P04637 Alternative ID of protein B (UniProt accession)
5 Aliases Interactor A uniprotkb:BRCA1_HUMAN(display_short)|entrezgene/locuslink:BRCA1(gene name) UniProt name, gene name and synonyms of protein A
6 Aliases Interactor B uniprotkb:P53_HUMAN(display_short)|entrezgene/locuslink:TP53(gene name) UniProt name, gene name and synonyms of protein B
7 Interaction Detection Methods psi-mi:"MI:0018"(two hybrid) Experimental detection methods (PSI-MI CV list)
8 Publication 1st Author - Not stored — always -
9 Publication Identifiers pubmed:20818382 PubMed IDs of supporting publications
10 Taxid Interactor A taxid:9606(Homo sapiens) NCBI taxonomy ID of protein A (always human)
11 Taxid Interactor B taxid:9606(Homo sapiens) NCBI taxonomy ID of protein B (always human)
12 Interaction Types psi-mi:"MI:0915"(physical association) Interaction types (PSI-MI CV list)
13 Source Databases BioGRID|IntAct Databases reporting this interaction
14 Interaction Identifiers - Cross-reference links per source database
15 Confidence Value hippie:0.85 HIPPIE confidence score (0–1)
16 Presence In Other Species HIPPIE ext taxid:10090(Mus musculus) Species in which the interaction is conserved
17 Gene Name Interactor A HIPPIE ext BRCA1 Entrez gene name of protein A
18 Gene Name Interactor B HIPPIE ext TP53 Entrez gene name of protein B
