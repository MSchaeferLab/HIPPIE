HIPPIE TAB Format

Each row represents one protein–protein interaction. Columns:

# Column Example Description
1 UniProt_A BRCA1_HUMAN UniProt entry ID of protein A
2 Entrez_A 672 NCBI Entrez Gene ID of protein A
3 UniProt_B P53_HUMAN UniProt entry ID of protein B
4 Entrez_B 7157 NCBI Entrez Gene ID of protein B
5 score 0.85 HIPPIE confidence score (0–1). ≥0.63 medium, ≥0.72 high confidence
6 sources BioGRID,IntAct Comma-separated source databases reporting this interaction
7 experiments coimmunoprecipitation,Two-hybrid Experimental detection methods used
8 pmids 20818382,19412169 PubMed IDs of supporting publications
